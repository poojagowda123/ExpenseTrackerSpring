
Expense Tracker Web Application

This project is a web-based Expense Tracker developed using Java Spring MVC. The application helps users manage their daily expenses by allowing them to add, view, update, and delete expense records.

Features:
User registration and login
Secure sign-in and sign-out using HttpSession
Add new expenses
View all expenses
Edit and delete expenses
Dashboard showing total expenses and number of records

Technologies Used:
Java
Spring MVC
JDBC Template
MySQL
JSP
HTML & CSS
Apache Tomcat

Project Structure:
Controller → Handles user requests
DAO → Performs database operations using JDBC Template
Model → Represents expense data
View (JSP) → User interface
Database → MySQL stores expense records

How to Run:
Import the project into Eclipse or IntelliJ.
Create a MySQL database named expense_tracker.
Update database credentials in the configuration file.
Run the project on Apache Tomcat server.
Open in browser:
http://localhost:8080/ExpenseTrackerMVC
